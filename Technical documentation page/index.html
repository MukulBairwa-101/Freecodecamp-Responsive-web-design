What is Go?

Go is a cross-platform, open source programming language Go can be used to create high-performance applications Go is a fast, statically typed, compiled language that feels like a dynamically typed, interpreted language

Go was developed at Google by Robert Griesemer, Rob Pike, and Ken Thompson in 2007 Go's syntax is similar to C++

What is Go Used For?

  1. Web development (server-side)
  2. Cloud-native development

Get started

To start using Go, you need two things:

A text editor, like VS Code, to write Go code

A compiler, like GCC, to translate the Go code into a language that the computer will understand

Syntax of go

A Go file consists of the following parts:

  • Package declaration
  • Import packages
  • Functions
  • Statements and expressions
package main
import("fmt)

func main() {

fmt.Println("Hello World!")

}

Fundamentals of go

Comments

A comment is a text that is ignored upon execution.

Go Single-line Comments

func main() {

// This is a comment

}

Go Multi-line Comments

func main() {

/* The code below will print Hello World

to the screen, and it is amazing */

}

Variables

Variables are containers for storing data values.

In Go, there are different types of variables, for example:>

  • int
  • float32
  • string
  • bool

Go Arrays

In Go, there are two ways to declare an array:

  • With the var keyword:

    var array_name = [length]datatype{values} // here length is defined

  • With the := sign:

    array_name := [length]datatype{values} // here length is defined
